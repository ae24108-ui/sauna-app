## 🚩概要

サウナ施設の情報を登録・閲覧し、各施設に対してレビューを投稿できるWebアプリです。
FlaskとSQLAlchemyを用いて開発し、Renderにデプロイしています。
施設一覧表示、施設登録、詳細表示、レビュー投稿の機能を実装しました。

---

## 🎯主な機能

- サウナ施設の一覧表示
- サウナ施設の新規登録
- 施設ごとの詳細表示
- 施設ごとのレビュー投稿
- 同じ名前の施設の重複登録防止

---

## ⚙️使用技術

- Flask
